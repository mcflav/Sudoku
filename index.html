SUDOKU

HOW TO PLAY:

Start game by clicking the Load Game button below. Enter your answer in each
red box above. You can ONLY complete ONE red box at a time. Click in the red box
(Do this also when the red box has been selected by default). It will display a lightblue background.
Enter your answer in the lightblue background and then click the Play button below.

To clear more than one box to enter new numbers select the boxes and click the Clear Items button.
A correct answer will turn the box green. Turn all red boxes green to win the game!
